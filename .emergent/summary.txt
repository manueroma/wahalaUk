    - **Microtransactions**: Pay £0.99 to skip the 1-hour chat wait. Virtual Roses for £0.10 each.
    - **Donation Button**: A button for users to donate.
    - **Affiliate Marketplace**: Charge affiliates £9.99/month to list their links.
- **User Profile & Features**:
    - Minimum of 3 pictures (max 10) required to use the app.
    - User profiles should include: Instagram handle, height (in cm), Looking for status.
    - Screenshot functionality should be disabled for privacy.
    - **Referral System**: Users get a unique referral link. If a new user signs up with the link and stays active for 24 hours, the referrer earns 20 virtual roses.
- **User Account**:
    - Permanent account deletion option.
    - 2FA for extra security.
    - Password reset via email/phone.
- **Admin Functionality**:
    - An admin account () with access to a dashboard.
    - The dashboard should show reported users.
    - Admin can view report details, issue warnings (3-strike system), or ban users.
- **Legal**: A Terms and Conditions page that users must accept, covering refund policies and the app's right to ban users.
- **Marketing Website**: A landing page at .
